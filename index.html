⚠️ Known Issues & Important Notes

🔍 Browser Zoom Limitation

Issue: Marker positions shift when using browser zoom (Ctrl+Plus/Minus or Ctrl+MouseWheel).

Why: Markers use pixel coordinates that don't scale proportionally with browser zoom.

✅ Best Practice:

Keep browser at 100% zoom while planning strategy.

How to Reset Zoom:

  • Press Ctrl+0 (Windows/Linux)
  • Press ⌘+0 (Mac)

Workflow:

  1. Reset zoom to 100% (Ctrl+0)
  2. Place all markers and create strategy
  3. Export your strategy (save progress)
  4. Now you can zoom for viewing

💡 Tips for Best Results

  • Use desktop/laptop for strategy planning
  • Export frequently to save progress
  • Use Import to restore saved strategies
  • Mobile is great for viewing, not detailed planning

📖 How To Use: Strategy Map Guide

🚀 Quick Start

  1. Load the page: your guild roster appears automatically in the left panel.
  2. Drag a player from the left panel onto the map to place them.
  3. Use the toolbar at the bottom to add objective markers, boss/tower/tree/goose icons, or draw arrows.
  4. When happy with the strategy, click 📥 Export to save a JSON file.
  5. Share the JSON with your team: they can 📤 Import it to see the same setup.

👥 Left Panel: Player Roster

  • Search bar: Filter players by name as you type.
  • Grouped by Team / All Players: Toggle between viewing players grouped into their teams or in a flat list.
  • Role filters: Click Tank, DPS, Healer, or Support to show only that role.
  • Drag to place: Drag any player card onto the map to place them. Their name label appears at the drop point.
  • Drag teams: You can drag a whole team group header to place all players together.
  • Unplace: Drag a placed marker off the map or click the × remove button to return them to the panel.
  • ⚙️ Manage Players: Edit, delete, or add players manually. Use the pencil icon to change name, role, team, or weapons.
  • ⚔️ Add Enemies: Places up to 5 generic enemy markers on the map for reference.

🔧 Bottom Toolbar

🎯
Objective Marker: Opens a picker with 8 types: Attack ⚔️, Danger 💀, Defend 🛡️, Hold ✋, Yes ✅, No ❌, and more. Click a type then click the map to place it. Draggable.
Boss: Places a boss marker. Click button then click anywhere on the map.
Towers (Blue / Red): Places a tower icon for your team (blue) or enemy (red).
Trees (Blue / Red): Same as towers but for tree/buff objectives.
Geese (Blue / Red): Same pattern, for goose objectives.
✏️
Draw Mode: Toggle freehand drawing on the map (useful for arrows and zones). All drawings can be undone.
↩️
Undo / Redo: Step back or forward through drawing history.
🧹
Clear Drawings: Removes all freehand drawings instantly.
❌
Clear Map: Removes all placed players and markers from the map. Does not affect the player roster.

📊 Importing Players from CSV / Excel

Click 📤 Import → Player Roster (.csv / .xlsx) and select your file. A preview will appear before you confirm.

Required column: Name
Optional columns: Role, Team, Weapon1, Weapon2

Column names are flexible: the importer recognises common variations (e.g. Player, IGN, Username for Name; Class, Type for Role; Squad, Group for Team).

Valid roles: Tank, DPS, Healer, Support: anything unrecognised defaults to DPS.

New teams: If your file contains team names that don't exist yet, they will be auto-created when you confirm.

💡 Tips & Best Practices

  • Keep your browser at 100% zoom (Ctrl+0) while placing markers to avoid position drift.
  • Export before closing the tab: the map does not auto-save between sessions.
  • The Export JSON file can be pasted into Discord so guildmates can load the exact same setup.
  • On mobile the page is best used for viewing a shared strategy, not detailed editing.
